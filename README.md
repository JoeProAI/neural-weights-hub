# Neural Weights Hub

A comprehensive platform for testing, comparing, and building applications with open weight models (gpt-oss-20b and gpt-oss-120b). Features instant sandbox deployment, model comparison tools, and collaborative development environments.

*Built by a neuroscientist with deep expertise in neural networks - from mouse sensory processing to cutting-edge AI models.*

## 🚀 Features

- **Model Showcase**: Interactive comparison between gpt-oss-20b and gpt-oss-120b
- **One-Click Sandboxes**: Deploy development environments instantly with Daytona
- **App Building Framework**: Templates and tools optimized for open weight models
- **Collaborative Development**: Share environments and work together on projects
- **Performance Analytics**: Real-time benchmarking and cost estimation

## 🛠 Tech Stack

- **Frontend**: Next.js 14 with TypeScript and Tailwind CSS
- **Authentication**: Firebase Auth
- **Database**: PostgreSQL with Prisma ORM
- **Sandbox Environments**: Daytona API integration
- **AI Models**: OpenAI gpt-oss-20b and gpt-oss-120b

## 🚦 Getting Started

1. **Clone and Install**
   ```bash
   git clone <repository-url>
   cd openai-app-builder
   npm install
   ```

2. **Environment Setup**
   ```bash
   cp .env.example .env.local
   # Add your API keys and configuration
   ```

3. **Run Development Server**
   ```bash
   npm run dev
   ```

4. **Open Browser**
   Navigate to [http://localhost:3000](http://localhost:3000)
